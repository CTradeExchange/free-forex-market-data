# Free Forex Market Data
Free Forex Quote API 免费的外汇贵金属行情API

简单好用的免费开源外汇贵金属行情API

## 功能特性：

- 免费开源

- 获取外汇实时行情

- 获取外汇实时多档盘口

- 获取外汇K线数据

- 获取贵金属实时行情

- 获取贵金属实时多档盘口

- 获取贵金属K线数据
